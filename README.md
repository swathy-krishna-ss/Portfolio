## 🚀 Technologies Used
This portfolio is built with modern web technologies to ensure a fast, responsive, and visually engaging experience:
- **Framework:** [Next.js](https://nextjs.org/) (React)
- **Styling:** [Tailwind CSS](https://tailwindcss.com/)
- **Design:** Custom-built academic theme with responsive micro-animations
- **Forms:** FormSubmit API for seamless contact form integration

## 🛠️ Running Locally

If you'd like to explore the code and run this project on your local machine:

1. **Clone the repository:**
   ```bash
   git clone https://github.com/swathy-krishna-ss/Portfolio.git
   cd Portfolio
   ```

2. **Install dependencies:**
   ```bash
   npm install --legacy-peer-deps
   ```

3. **Start the development server:**
   ```bash
   npm run dev
   ```

4. **View the site:**
   Open [http://localhost:3000](http://localhost:3000) in your browser.
